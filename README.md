## 🔥 Featured Work

### TiMem — Temporal-Hierarchical Memory for Long-Horizon Conversational Agents

**Role:** Algorithm Co-founder / Lead of Agent Memory Direction
**Type:** Research + Open Source + Product Delivery

TiMem is a memory & cognition engine for LLM agents, designed for:

* long-horizon conversations
* personalized interaction
* memory consolidation and retrieval
* robust reasoning over evolving user context

### What I led

* End-to-end ownership: **problem framing → memory mechanism design → MVP implementation → evaluation pipeline → iteration**
* Core memory algorithms: memory structure / consolidation / retrieval / selection
* Integration with **Agent workflow**, **RAG**, **tool use**, and **self-evolving modules**
* Temporal-semantic aware hierarchical memory framework for stable long-term interaction

### Outputs

* 📄 Paper (arXiv): [TiMem: Temporal-Hierarchical Memory Consolidation for Long-Horizon Conversational Agents](https://arxiv.org/abs/2601.02845)
